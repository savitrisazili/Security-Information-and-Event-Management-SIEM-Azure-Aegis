# Presentation Folder

This folder contains the presentation slides for the SIEM Wazuh Lab project developed by Azure Aegis (Blue Team 7).

## Presentation Contents

- Project overview and objectives
- Wazuh SIEM implementation
- Windows, Ubuntu, and Web Server agent integration
- Attack simulation scenarios
- Detection rules and monitoring process
- Dashboard visualization and Telegram alerts
- Testing results and troubleshooting
- Final conclusions of the project

The presentation summarizes the overall implementation, monitoring architecture, attack detection workflow, and security analysis performed during the project.
